# WeatherApp

A simple static Weather demo app (HTML/CSS/JS).

## Description
- Minimal front-end project showing a weather UI. No backend required.

## Files
- `config.js` — configuration
- `index.html` — main page
- `script.js` — UI logic
- `style.css` — styles

## Prerequisites
- Modern web browser (Chrome, Firefox, Edge)
- (Optional) Python for a local static server

## Run locally
- Open `index.html` directly in your browser, or serve the folder:

```bash
# from the repository root (one level above the WeatherApp folder)
python -m http.server 8000
# then open:
# http://localhost:8000/WeatherApp/index.html
```

## Development
- Edit `index.html`, `script.js`, and `style.css` and refresh the browser.
